## 十五、门户管理

> **功能路径**：`/portal` 模块
> **权限要求**：租户管理员（tenant_admin）
> **套餐要求**：基础版及以上

### 15.1 门户页面

#### 15.1.1 功能描述
可视化门户页面配置，支持拖拽布局、添加组件、设置主题样式。

#### 15.1.2 数据表
| 字段 | 类型 | 说明 |
|------|------|------|
| page_id | UUID | 主键 |
| tenant_id | UUID | 所属租户 |
| page_name | VARCHAR(50) | 页面名称 |
| page_code | VARCHAR(30) | 页面编码 |
| page_type | ENUM | dashboard / landing / custom |
| layout_config | JSONB | 布局配置 |
| components | JSONB | 组件列表 |
| theme | JSONB | 主题样式 |
| is_default | BOOLEAN | 是否默认首页 |
| status | ENUM | draft / published / archived |
| created_at | TIMESTAMPTZ | 创建时间 |
| updated_at | TIMESTAMPTZ | 更新时间 |

#### 15.1.3 API接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/tenants/{tenant_id}/portal/pages` | 页面列表 |
| POST | `/api/v1/tenants/{tenant_id}/portal/pages` | 创建页面 |
| GET | `/api/v1/tenants/{tenant_id}/portal/pages/{id}` | 页面详情 |
| PUT | `/api/v1/tenants/{tenant_id}/portal/pages/{id}` | 更新页面 |
| DELETE | `/api/v1/tenants/{tenant_id}/portal/pages/{id}` | 删除页面 |
| POST | `/api/v1/tenants/{tenant_id}/portal/pages/{id}/publish` | 发布页面 |
| GET | `/api/v1/tenants/{tenant_id}/portal/pages/{id}/preview` | 预览页面 |
| PUT | `/api/v1/tenants/{tenant_id}/portal/pages/{id}/set-default` | 设为默认页面 |

#### 15.1.4 按钮定义
**页面路径**: `/portal/pages`
**布局**: 列表页

| 位置 | 按钮名称 | 动作 | 权限 |
|------|----------|------|------|
| 操作栏（左） | 「搜索」 | 展开筛选面板 | all |
| 操作栏（右） | 「刷新」 | 刷新列表 | all |
| 操作栏（右） | 「新建页面」 | 打开全屏模态 | tenant_admin |
| 表格行 | 「编辑」 | 打开设计器 | tenant_admin |
| 表格行 | 「预览」 | 新窗口打开 | all |
| 表格行 | 「发布」 | 确认弹窗 | tenant_admin |
| 表格行 | 「设默认」 | 确认弹窗 | tenant_admin |
| 表格行 | 「删除」 | 删除确认弹窗 | tenant_admin |
