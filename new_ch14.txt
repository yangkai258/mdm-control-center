## 十四、低代码CRUD

> **功能路径**：`/lowcode` 模块
> **权限要求**：租户管理员（tenant_admin）
> **套餐要求**：专业版及以上

### 14.1 标准CRUD

#### 14.1.1 功能描述
通用 CRUD 生成器，通过配置数据模型自动生成标准列表页、详情页、表单页和对应 API。

#### 14.1.2 数据表
| 字段 | 类型 | 说明 |
|------|------|------|
| model_id | UUID | 主键 |
| tenant_id | UUID | 所属租户 |
| model_name | VARCHAR(50) | 模型名称 |
| model_code | VARCHAR(30) | 模型编码（英文，唯一） |
| table_name | VARCHAR(50) | 对应数据库表名 |
| fields | JSONB | 字段定义数组 |
| list_config | JSONB | 列表页配置 |
| form_config | JSONB | 表单页配置 |
| api_config | JSONB | API 配置 |
| status | ENUM | draft / generated / published |
| created_at | TIMESTAMPTZ | 创建时间 |

#### 14.1.3 API接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/tenants/{tenant_id}/lowcode/models` | 模型列表 |
| POST | `/api/v1/tenants/{tenant_id}/lowcode/models` | 创建数据模型 |
| GET | `/api/v1/tenants/{tenant_id}/lowcode/models/{id}` | 模型详情 |
| PUT | `/api/v1/tenants/{tenant_id}/lowcode/models/{id}` | 更新模型 |
| DELETE | `/api/v1/tenants/{tenant_id}/lowcode/models/{id}` | 删除模型 |
| POST | `/api/v1/tenants/{tenant_id}/lowcode/models/{id}/generate` | 生成 CRUD 代码 |
| POST | `/api/v1/tenants/{tenant_id}/lowcode/models/{id}/preview` | 预览生成结果 |

#### 14.1.4 按钮定义
**页面路径**: `/lowcode/models`
**布局**: 列表页

| 位置 | 按钮名称 | 动作 | 权限 |
|------|----------|------|------|
| 操作栏（左） | 「搜索」 | 展开筛选面板 | all |
| 操作栏（右） | 「刷新」 | 刷新列表 | all |
| 操作栏（右） | 「新建模型」 | 打开全屏模态 | tenant_admin |
| 表格行 | 「编辑」 | 打开全屏模态 | tenant_admin |
| 表格行 | 「生成代码」 | 确认弹窗后生成 | tenant_admin |
| 表格行 | 「预览」 | 打开抽屉 | tenant_admin |
| 表格行 | 「删除」 | 删除确认弹窗 | tenant_admin |
