## 错题本（2026-04-06）

### 1. Vite Preview 懒加载 Chunk 不稳定 ⚠️
- **现象**：点击多个菜单后页面变空白，刷新恢复
- **根因**：`vite preview` 的静态文件服务在快速连续请求懒加载 chunk 时会失败
- **教训**：开发阶段用 `npm run dev`（dev server），不要用 `vite preview`
- **预防**：preview 适合验证生产构建，不适合开发测试

### 2. 父路由必须有组件 ⚠️⚠️
- **现象**：子路由页面空白，控制台无错误
- **根因**：父路由只有 `redirect`，没有 `component`，子路由无处渲染
- **教训**：所有父路由都要指定 `component: ParentLayout`
- **预防**：路由设计时检查每个路由的 component 字段

### 3. 视图文件和路由必须同时存在 ⚠️
- **现象**：菜单点击后空白
- **根因**：路由指向的 `.vue` 文件不存在
- **教训**：新建路由时同时创建对应的视图文件
- **预防**：路由注册后立即验证页面能打开

### 4. keep-alive 缓存导致 Component undefined ⚠️
- **现象**：页面导航后有时显示空白
- **根因**：`keep-alive` 缓存的组件再次激活时 `Component` 为 `undefined`
- **教训**：`v-if="Component"` 守卫是必须的
- **预防**：使用 keep-alive 时做好 undefined 兜底

### 5. Git 浅克隆问题（历史教训）
- **现象**：GitHub 有文件但本地没有
- **根因**：用了 `--depth=1`
- **教训**：永远全量 clone 和 fetch
- **预防**：已记录到 TOOLS.md

### 6. locale 键必须与路由完全匹配 ⚠️
- **现象**：菜单显示 `menu.xxx.yyy` 这种 key
- **根因**：locale 文件缺少对应翻译
- **教训**：新增路由时同步添加 locale 键
- **预防**：建立 locale 键和路由的对应检查表
