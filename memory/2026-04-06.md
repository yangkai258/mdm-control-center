## 技术笔记

### 菜单系统架构
- `menuFromServer: false` → 从本地 `appRoutes` 读取路由生成菜单
- `menuFromServer: true` → 从后端 `/auth/menu` API 获取菜单
- 本地路由：所有顶级路由（有 `meta.locale`）会显示为菜单项
- 菜单组件：`useMenuTree()` → `appClientMenus` → `appRoutes`
