## 技术问题与解决

### 问题1：PetController Redis nil panic
- **原因**：`PetController{DB: db}` 未传 Redis 客户端
- **解决**：main.go 中改为 `PetController{DB: db, Redis: redisClient}`

### 问题2：行为记录 500 错误
- **原因**：`PetBehaviorAction` 表未在 AutoMigrate 中注册
- **解决**：在 ActionLearning AutoMigrate 行添加 `&models.PetBehaviorAction{}`

### 问题3：配额 API 路径错误
- **原因**：控制器注册为 `/quota`，前端调用 `/quotas`
- **状态**：路径错误，确认正确路径即可

### 问题4：设备配对完全缺失
- **原因**：PRD 定义了但从未实现
- **解决**：从零创建 device_pairing.go + device_pairing_controller.go
