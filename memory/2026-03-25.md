## 经验教训

1. Docker cp无法覆盖运行中容器的文件，必须先stop或用sleep容器中转
2. docker commit + 新容器部署是绕过网络问题的好方法
3. PowerShell变量在嵌套docker exec中会提前解析，需要用文件方式传参
4. 调试代码要及时清理，避免影响正常功能
