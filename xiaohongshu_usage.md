# 小红书技能使用指南

## 安装状态
✅ 小红书技能已成功安装到OpenClaw
✅ Cookies已配置并验证有效（有效期至2027年）

## 可用功能
1. **搜索内容** - 按关键词搜索小红书笔记
2. **首页推荐** - 获取首页推荐列表
3. **帖子详情** - 获取笔记内容、评论、互动数据
4. **发表评论** - 在笔记下发表评论
5. **用户主页** - 获取用户资料和笔记列表
6. **热点跟踪** - 自动生成话题分析报告
7. **长图导出** - 将帖子导出为带注释的JPG长图
